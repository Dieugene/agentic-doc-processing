# Agentic Document Processing System

> **Статус:** В разработке
>
> Система для глубокого анализа больших документов с помощью AI-агентов.

## О проекте

Система предназначена для интеллектуальной обработки и анализа документов (сотни страниц) с использованием многоагентной архитектуры. Оптимизирована для работы с нормативными актами, годовыми отчётами компаний и другими сложными документами.

## Ключевые возможности

- **Извлечение информации** — ответы на вопросы по содержанию документа
- **Анализ нестыковок** — выявление внутренних противоречий
- **Сравнение документов** — включая проверку соответствия нормативным требованиям
- **Формализация регламентов** — генерация BPMN-диаграмм и правил

## Архитектура

Система использует трёхслойную архитектуру данных:

1. **Структурный скелет** — граф физической структуры документа (разделы, таблицы, приложения)
2. **Навигационный индекс** — связь структуры с темами (что и где раскрывается)
3. **Тематические снэпшоты** — специализированные агенты с накопленным контекстом по темам

Все агенты построены на фреймворке **SGR (Schema-Guided Reasoning)**, что позволяет использовать дешёвые модели (Claude Haiku, GPT-4o-mini, локальные) без потери качества.

## Основные агенты

| Агент | Назначение |
|-------|------------|
| **Диспетчер** | Единая точка входа, маршрутизация запросов |
| **Индексатор** | Построение навигационного индекса документа |
| **Нормализатор** | Создание общей таксономии тем |
| **Снэпшот-агенты** | Ответы на вопросы по конкретным темам |
| **Агент уровня документа** | Комплексный анализ (BPMN, структурирование) |

## Технологии

- **Фреймворк агентов:** [SGR Agent Core](https://vamplabAI.github.io/sgr-agent-core/)
- **Модели:** Дешёвые модели (Haiku, GPT-4o-mini) + сильные модели для holisitic-анализа
- **LLM Gateway:** Централизованный доступ с батчингом и rate limit control
